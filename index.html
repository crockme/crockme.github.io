JavaScript介绍

JavaScript是基于对象的事件驱动型脚本语言

  • 基于对象，有别于面向对象
  • 脚本语言，寄生于HTML页面，用来实现对网页中HTML标签的操作
  • 事件驱动，网页中发生的动作触发JS代码的执行

1.JS可以干什么？

  • JS可以动态修改页面HTML标签的属性
  • JS可以改变页面HTML标签的样式
  • JS可以动态改变文档的HTML标签结构
  • JS可以实现页面与后台数据的动态交互（提交数据，获取数据）

2.作为Java程序员，我们如何学习JS？

  • JS基础语法
  • JS内置对象
    • String
    • Array
    • Math
    • RegExp
  • JS BOM（Browser Object Model）
    • window
      • document
      • location
      • history
      • navigator
      • screen
  • JS DOM（Document Object Model）节点操作

3.获取页面元素的几种方式：

  • var tag = document.getElementById(“id”);
  • var tags = document.getElementsByName(“name”);
  • var tags = document.getElementsByTagName(“tagName”);
  • var tags = document.getElementsByClassName(“class”);

网页设计

网页，可以通过互联网传输、在浏览器打开呈现视图界面、后缀名为html/htm的文件

  • 一个网页就是一个后缀名为.html的文件

  • 文件可以在网络传输

  • 文件可以通过浏览器打开

1.HTML

HTML（HyperText Markup Language）超文本标记语言，用于向浏览器传达显示意图。

  • 超文本：以文本的形式在浏览器中显示超出文本范畴的视图（图片，控件，声音，视频）

  • 标记：没有逻辑性，一个标记就代表一种视图效果

  • 标签：

    • 单标签，只有一部分组成的标签。语法结构：

    • 双标签，有两部分组成的标签，在头标签和尾标签之间可以添加特定内容。

      语法结构：

  • 属性：在标签中添加的键值对，用于设置/改变标签的特性。

  • 网页的基本结构：

    • 通过声明当前HTML文档遵守的规范
    • HTML文档有一对根标签：
    • html根标签下有两对子标签：head和body
      • 用于设置当前网页标题、编码等属性相关的内容；
      • 用于承载当前网页的内容

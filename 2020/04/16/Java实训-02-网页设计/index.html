网页设计

网页，可以通过互联网传输、在浏览器打开呈现视图界面、后缀名为html/htm的文件

  • 一个网页就是一个后缀名为.html的文件

  • 文件可以在网络传输

  • 文件可以通过浏览器打开

1.HTML

HTML（HyperText Markup Language）超文本标记语言，用于向浏览器传达显示意图。

  • 超文本：以文本的形式在浏览器中显示超出文本范畴的视图（图片，控件，声音，视频）

  • 标记：没有逻辑性，一个标记就代表一种视图效果

  • 标签：

    • 单标签，只有一部分组成的标签。语法结构：

    • 双标签，有两部分组成的标签，在头标签和尾标签之间可以添加特定内容。

      语法结构：

  • 属性：在标签中添加的键值对，用于设置/改变标签的特性。

  • 网页的基本结构：

    • 通过声明当前HTML文档遵守的规范
    • HTML文档有一对根标签：
    • html根标签下有两对子标签：head和body
      • 用于设置当前网页标题、编码等属性相关的内容；
      • 用于承载当前网页的内容
